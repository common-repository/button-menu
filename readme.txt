Button Menu is an easy to use WordPress menu plugin to show your navigation menu anywhere on a page as buttons using a shortcode.

== Description ==

Show your WordPress navigation menus anywhere on any page using a shortcode as buttons. Supports up to 3 levels of menus. Beautiful button styles and animations.

[Support, Bug Report, Feature Requests](https://www.dna88.com/support-free/) | [Button Menu Pro Demo](https://dev.dna88.com/button-menu/) | <strong>[Upgrade to Button Menu Pro!](https://www.dna88.com/product/button-menu/)</strong>

Button menu displays the navigation menus you created in Appearance->Menus in an intuitive and useful way.
Place the following shortcode on your page or post where you want to show the Button menu. [qc-button-menu menu='menu_name']

Copy the menu name exactly as you set under Appearance->Menus->Your selected menu.

Show your WordPress navigation menus anywhere on any page as buttons easily using a shortcode. Button menu pro supports unlimited sub menu levels with icons, animations and complete control over the colors of the individual icons.

== Installation ==

Unzip and Upload the wp-voice folder to /wp-content/plugins/
Activate the plugin through the 'Plugins' menu in WordPress
Navigate to Button menu setting in wp-admin and configure
Place the following shortcode on your page or post where you want to show the Button menu
[qc-button-menu menu='menu_name']
Copy the menu name exactly as you set under Appearance->Menus->Your selected menu.
